# FE-API-001 — Créer types base (Property, Booking, Negotiation, Payment, User)

## Priorité
P0

## Délai estimé
1.5d

## Dépendances
Aucune

## Objectif
Définir les types TypeScript fondamentaux pour garantir une base de données cohérente et réutilisable dans toute l’application.

## Tâches
- Définir les interfaces `Property`, `PropertyDetail`, `Booking`, `Negotiation`, `Payment`, `User`.
- Créer les types de pagination et de réponses API standardisées.
- Définir les types pour filtres et recherche.
- Documenter les champs critiques (ex. dates, currency).

## Critères d’acceptation
- Types compilent sans erreurs TypeScript.
- Types exportés et réutilisables dans tout le projet.
- Documentation inline des propriétés importantes.
